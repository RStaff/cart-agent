# Shopifixer Daily Execution Checklist V1

## Daily Send Target

- Send 20 first-touch messages

## Follow-Up Block

- Review active conversations
- Send first follow-ups
- Send second follow-ups where appropriate

## Response Handling Block

- Reply to interested prospects
- Send audit link only after interest
- Log objections and questions

## Close Attempts

- Send the $99 close where the conversation is warm
- Move paid prospects into delivery

## End-Of-Day Metrics

- sent
- responses
- audits
- closes
