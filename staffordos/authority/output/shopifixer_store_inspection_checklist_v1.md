# SHOPIFIXER STORE INSPECTION CHECKLIST V1

## Purpose

Find one specific friction issue suitable for a $950 ShopiFixer Fix Sprint.

## Inspect These Areas

1. Homepage clarity
2. Product page clarity
3. Add-to-cart visibility
4. Cart page clarity
5. Checkout path clarity
6. Mobile usability
7. Trust signals
8. Shipping/returns clarity
9. Product description clarity
10. Visual hierarchy

## Good Fix Candidate

A good first fix is:

- visible
- specific
- explainable
- likely to reduce hesitation
- feasible in one sprint
- provable with before/after evidence

## Bad Fix Candidate

Avoid:

- full redesign
- vague conversion optimization
- deep custom app work
- unmeasurable backend cleanup
- anything requiring broad merchant approval

## Output Needed

Choose one issue and record:

- affected page URL
- observed friction
- why it matters
- scoped fix
- before evidence needed
